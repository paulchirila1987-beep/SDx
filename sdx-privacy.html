Privacy Policy

Last updated: March 2026

✓ SDx does not collect, transmit, or store any personal data on external servers.

Overview

SDx — Speed Dial ("the extension") is a browser extension that replaces your new tab page with a customizable speed dial. This privacy policy explains what data the extension accesses and how it is used.

Data we collect

None. SDx does not collect any personal information, usage statistics, analytics, or telemetry of any kind.

The extension does not transmit any data to any server controlled by the developer or any third party.

Local storage

SDx stores your settings and speed dial links locally in your browser using the standard browser localStorage API. This data:

  • Never leaves your device
  • Is only accessible to the extension itself
  • Can be deleted at any time by clearing your browser data or uninstalling the extension
  • Can be exported and imported by you as a JSON file (backup feature)

Favicon loading

To display website icons for your speed dial tiles, the extension requests favicons from two public APIs:

  • DuckDuckGo Favicon API — icons.duckduckgo.com
  • Google Favicon API — www.google.com/s2/favicons (fallback only)

These requests include only the domain name of the website whose icon is being fetched (e.g. youtube.com). No personal data, browsing history, or identifiers are included. These services are governed by their own privacy policies.

Search

If you enable the optional search bar and perform a search, your query is sent directly to the search engine you selected (Google, DuckDuckGo, or Brave). SDx does not intercept, log, or process your search queries in any way.

Permissions

SDx requests only the storage permission, which allows saving your settings locally in the browser. No other browser permissions are requested or used.

Third-party services

SDx does not integrate any advertising networks, analytics platforms, crash reporting services, or social media trackers.

Changes to this policy

If this privacy policy changes in a future version of the extension, the updated policy will be published at this URL and the "last updated" date above will be revised.
